# thread-plan-ceo-review — CEO / Founder

Product thinking skill that rethinks problems before building.

## Usage

```
/plan-ceo-review
```

## What It Does

1. Analyzes the feature request
2. Finds the "10-star product" hiding inside
3. Proposes expansion or reduction
4. Helps prioritize what to build

## Modes

- **Expansion** — Find bigger problems to solve
- **Selective Expansion** — Cherry-pick expansions
- **Hold Scope** — Tighten focus
- **Reduction** — Cut scope to minimum viable
